# Простой чат
### Установка
Для установки необходимо скопировать последнюю ревизию из репозитория.
Дополнительная настройка или установка не требуется. Приложение не имеет зависимых модулей.

После того, как приложение будет скопировано, необходимо перейти в его корневую директорию и запустить nodejs сервер.
```sh
node index.js
```
Далее, необходимо перейти в браузере по нижеприведенному адресу, чтобы открыть приложение. 
```sh
127.0.0.1:8088
```
При разработке использовались:
- Node.js 4.4.5
- NPM 2.15.5
- Google Chrome 50
- Firefox 46
